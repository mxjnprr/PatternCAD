# PatternCAD Keyboard Shortcuts Reference

## Global Tool Shortcuts (MainWindow)
| Key | Tool/Action | Status |
|-----|-------------|--------|
| Z | Select Tool | ✅ Used |
| D | Polyline Tool (Draw) | ✅ Used |
| A | Add Point on Contour Tool | ✅ Used |
| R | Rotate Tool | ✅ Used |
| M | Mirror Tool | ✅ Used |
| Shift+S | Scale Tool | ✅ Used |
| S | Seam Allowance Tool | ✅ Used |
| N | Notch Tool | ✅ Used |
| +/= | Zoom In | ✅ Used |
| - | Zoom Out | ✅ Used |
| Backspace | Delete (alternative) | ✅ Used |
| Delete | Delete objects/vertices | ✅ Used |

## Tool-Specific Shortcuts (within tool context)
| Key | Context | Action |
|-----|---------|--------|
| T | SelectTool, AddPointOnContourTool | Toggle vertex type (Sharp ↔ Smooth) |
| G | SelectTool | Grab/move selected vertex |
| L | SelectTool | Lock segment length |
| Tab | SelectTool, PolylineTool, ScaleTool, RotateTool | Dimension input |
| Space | SelectTool | Grab/place object |
| H | MirrorTool | Horizontal mirror |
| V | MirrorTool | Vertical mirror |
| C | MirrorTool | Copy mode toggle |
| U | ScaleTool | Uniform scale toggle |
| 1/2/3 | NotchTool | Notch styles (V-Notch/Slit/Dot) |
| E | MatchPointTool | Edit label |
| Shift+Click | MatchPointTool | Link match points |
| Escape | All tools | Cancel/deselect |
| Enter/Return | Various | Confirm action |
